Welcome

This is an interactive view of the ResearchShop extraction pipeline. Each box is a function or pipeline domain in the Python code. Connections show data flow.

How to use

1. Click any node to see its description, inputs, outputs, and config flags.

2. Generate a trace by running the pipeline with --trace-pmid <PMID>.

3. Load the trace file with the button above to see actual data flowing through each domain for that paper.

4. Or try the demo — load the bundled demo_trace.json to see a populated view.
