\begin{abstract}
We present \textcolor{red}{\textbf{tool name}}, a free, open-source desktop application that automates the extraction of gene-disease associations from full-text biomedical literature. The system employs a hybrid architecture combining Named Entity Recognition (PubTator3) with Large Language Model extraction (Google Gemini Flash) to achieve high recall while maintaining precision through a multi-layer validation framework. This framework includes gene symbol verification against the HGNC database (44,943 genes), grounding checks that verify extracted genes appear in source text, HGVS-compliant variant validation, and citation cross-referencing using dense sequence matching. The application runs entirely on consumer hardware with no server infrastructure; users provide their own Gemini API key.
