\title{ResearchShop Desktop: A Desktop Application for Auditable Gene and Variant Extraction from Biomedical Literature Using Hybrid NER--LLM Architecture}

\begin{abstract}
We present ResearchShop Desktop, a free, open-source desktop application that assists biomedical researchers with structured extraction of gene and variant associations from PubMed literature. The system combines PubTator3 Named Entity Recognition, open-access full-text parsing from PubMed Central and Europe PMC, and Google Gemini Flash extraction behind a validation-oriented pipeline. Its safeguards include HGNC gene symbol verification, source-text grounding checks, HGVS-style variant pattern validation, citation cross-referencing using dense sequence matching, and explicit provenance fields for each output row. The application runs on consumer hardware with no server infrastructure; users provide their own Gemini API key and NCBI Entrez email.

The contribution of ResearchShop is a reproducible desktop workflow for paper discovery, open-access full-text acquisition, candidate gene discovery, LLM-assisted structured extraction, validation, and CSV/XLSX export. The software is packaged for macOS, Windows, and Linux, includes an offline test suite for core parser and validation behavior, and persists audit artifacts that help users inspect candidate provenance, gate drops, confidence scores, and extraction failures. ResearchShop is intended as a research acceleration tool; extracted associations require expert review before use in publications, systematic reviews, or clinical contexts.
\end{abstract}
