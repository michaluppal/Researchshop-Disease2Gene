\section{Conclusions}
\label{sec:conclusions}

We have presented ResearchShop Desktop, an open-source desktop application for auditable gene and variant extraction from PubMed literature. The hybrid NER--LLM architecture combines PubTator3 candidate discovery, deterministic HGNC candidate scanning, Gemini Flash structured extraction, open-access full-text parsing, and a multi-layer validation framework. HGNC verification, source-text grounding, variant pattern matching, citation cross-referencing, confidence scoring, and candidate audit artifacts provide defence-in-depth against common LLM extraction failures.

The software is freely available under the MIT licence as a cross-platform desktop application. It requires no server infrastructure; users provide their own Gemini API key and NCBI Entrez email. The project includes offline tests for parser, validation, tracing, and output behavior, plus release scripts for macOS, Windows, and Linux installers.

\paragraph{Future work.} Planned enhancements include: (1)~claim-level verification for free-text findings, separating evidence checking for each extracted assertion rather than validating only citation strings; (2)~LLM-assisted PubMed query construction with HGNC alias awareness and explicit diffing against the user's original query; (3)~ClinicalTrials.gov integration for pharmacogenomics and translational workflows where trial records provide structured gene--drug context; (4)~cross-run aggregation and notebook-style result exploration so users can compare repeated runs and adjudicate uncertain rows; (5)~expanded variant extraction coverage for complex structural variants and non-HGVS notation; (6)~specialised handling of immunoglobulin and HLA nomenclature, whose IMGT and IPD conventions conflict with HGNC identifiers; and (7)~multi-paper relationship inference to aggregate gene--disease evidence across a corpus and identify consensus associations.
