\section{Conclusions}
\label{sec:conclusions}

We have presented ResearchShop, an open-source desktop application for auditable gene and variant extraction from PubMed literature. The hybrid NER--LLM architecture combines PubTator3 candidate discovery, deterministic HGNC candidate scanning, configurable LLM-based structured extraction, open-access full-text parsing, and a multi-layer validation framework. HGNC verification, source-text grounding, variant pattern matching, citation cross-referencing, confidence scoring, and candidate audit artifacts provide defence-in-depth against common LLM extraction failures.

The application requires no ResearchShop server infrastructure; users provide their own Gemini API key and NCBI Entrez email for direct calls to external services. The project includes offline tests for parser, validation, tracing, and output behavior, plus release scripts for macOS, Windows, and Linux installers.

\paragraph{Future work} Planned enhancements include claim-level verification for free-text findings, LLM-assisted PubMed query construction with HGNC alias awareness, ClinicalTrials.gov integration for pharmacogenomics workflows, local or self-hosted model backends for fully private extraction when papers are supplied locally, cross-run aggregation for adjudicating uncertain rows, broader structural-variant and HLA/immunoglobulin nomenclature handling, and multi-paper relationship inference to identify consensus gene--disease associations.
